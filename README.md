# Cloud Computing Project 1 - Personal Portfolio Website

This project is a responsive personal portfolio website developed as part of the DecodeLabs Cloud Computing Internship Project 1.

The website is hosted using **Amazon AWS S3 Static Website Hosting**, making it accessible globally without using a traditional web server.


## 🌐 Live Website

http://sid-cloud-computing.s3-website.eu-north-1.amazonaws.com



## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- Amazon AWS S3


## ✨ Features

- Responsive design
- Smooth animations
- Typing effect
- Profile section
- Skills section
- Projects section
- Achievements section
- Resume download option
- Social media links

## 📁 Project Structure

```text
cloud_computing_1
│
├── index.html

├── script.js
├── profile.jpeg
├── resume.pdf

```
